# Health Informatics Project

## Summary

This project explores how artificial intelligence can be used in healthcare to improve patient outcome prediction.

The goal is to design a model that predicts hospital readmission risk using patient data such as age, diagnosis history, lab results, and previous admissions.

Machine learning methods like logistic regression or neural networks could help identify high-risk patients early.

The expected outcome is to support hospitals in improving care quality and reducing unnecessary readmissions.
